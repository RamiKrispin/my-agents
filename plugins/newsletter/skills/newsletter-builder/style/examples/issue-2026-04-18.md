## Open Source of the Week

This week’s focus is on the **smooth** project. I learned about this library after coming across **Leonidas Tsaprounis**‘s **[post](https://www.linkedin.com/feed/update/urn:li:activity:7449412504641916928)** about the release of the Python version. The **smooth** package implements **Single Source of Error (SSOE)** state-space models for forecasting and time series analysis and is available for both R and Python, and is based on the work of **Ivan Svetunkov**. One of its key features is the **ADAM (Augmented Dynamic Adaptive Model)** framework, which provides a flexible and powerful approach to time series forecasting.

More details about the ADAM forecasting approach are available in Ivan Svetunkov’s book, *Forecasting and Analytics with the Augmented Dynamic Adaptive Model (ADAM)*. Additional information can be found in the Book of the Week section below.

Project repo: **[https://github.com/config-i1/smooth](https://github.com/config-i1/smooth)**

### **Key Features**

- Implements the **ADAM framework**, combining ideas from ETS, ARIMA, regression, and state space models in a single interface

- Supports classical forecasting methods, including **exponential smoothing, SARIMA, and moving averages**

- Handles **multiple seasonal patterns** and external regressors for complex business time series

- Includes tools for **intermittent demand forecasting**, useful for inventory and supply chain planning

- Provides simulation, diagnostics, residual analysis, and forecast accuracy evaluation tools

- Available in **R** with an emerging **Python interface**, expanding accessibility across ecosystems

- Well-suited for production forecasting workflows, academic research, and advanced experimentation with forecasting models

More details are available in the project [documentation](https://github.com/config-i1/smooth/wiki).

## Book of the Week

Following the open-source of the week, this week’s book focus is on the ADAM models: Forecasting and Analytics with the Augmented Dynamic Adaptive Model (ADAM) by **[Ivan Svetunkov](https://www.linkedin.com/preload/?_bprMode=vanilla#)**. The book introduces the **ADAM (Augmented Dynamic Adaptive Model)** framework — a unified state space approach that combines and extends classical forecasting methods such as ETS, ARIMA, and regression models. The book presents a flexible modeling system that supports multiple seasonalities, non-normal distributions, intermittent demand, dynamic regression, and custom loss functions. It blends statistical theory with practical implementation using the **smooth** package in R, making it a valuable bridge between academic forecasting methods and real-world business applications.

- **The ADAM framework** — understanding how ETS, ARIMA, and regression are unified within a single modeling structure.

- **Exponential smoothing and ETS models** — level, trend, seasonality, damped trends, and state space forecasting methods.

- **ARIMA modeling** — autoregressive and moving average models within the ADAM framework.

- **Dynamic regression models** — incorporating explanatory variables and time-varying parameters into forecasts.

- **Multiple seasonalities** — handling complex seasonal patterns such as hourly, weekly, or yearly cycles.

- **Intermittent demand forecasting** — modeling sparse or zero-heavy demand series common in retail and supply chains.

- **Distributional forecasting** — using non-normal distributions and producing probabilistic forecasts.

- **Model selection and diagnostics** — automatic order selection, variable selection, outlier detection, and uncertainty estimation.

- **Practical forecasting workflows** — implementing ADAM models in R with the smooth and greybox packages.

This book is ideal for experienced analysts, data scientists, forecasters, economists, and researchers who already understand statistics or regression and want a modern, rigorous framework for advanced univariate forecasting and business analytics.

Thanks to the author, the book has an open online [version](https://openforecast.org/adam/).

A hard copy of the 1st edition is available for purchase on the **[publisher’s website](https://www.routledge.com/Forecasting-and-Analytics-with-the-Augmented-Dynamic-Adaptive-Model-ADAM/Svetunkov/p/book/9781032590370)** and on **[Amazon](https://www.amazon.com/Forecasting-Analytics-Augmented-Dynamic-Adaptive/dp/1032590378)**.
