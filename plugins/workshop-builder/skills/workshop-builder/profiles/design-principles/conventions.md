# Conventions — `design-principles`

This profile uses the standard workshop conventions in
`../../references/conventions.md` and `../../references/structure.md` as-is
(`NN_topic_name/` folders, slides under `slides/`, no scripts, opt-in topic
READMEs).

No per-profile overrides. If a target repo needs different rules, document them
in that repo's `spec/naming-convention.md`, which takes precedence.
