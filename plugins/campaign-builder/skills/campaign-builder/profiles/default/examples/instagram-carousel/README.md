frontmatter, no commentary. See `../README.md` for the full convention.

This folder defines the **instagram-carousel** voice for campaign-builder. The skill reads
every `*.md` file here when drafting a campaign post for this platform. If
the folder is empty, the skill falls back to `../../../formats/instagram-carousel*.md`
and warns. Populate over time to sharpen the platform-specific voice.
