# Voice examples

Real issues of **Rami's Data Newsletter**, used as few-shot anchors for voice and
structure. `../voice-guide.md` is derived from these.

These were pulled automatically from the public Substack archive. To refresh or
add more:

```bash
python3 ../scripts/fetch-examples.py 8   # from the newsletter-builder/ dir
```

(The fetcher skips short one-off posts like course promos and workshop
announcements; it keeps standard three-section issues.)

You can also hand-add or hand-edit `.md` files here — one issue per file
(`issue-YYYY-MM-DD.md`). Pick the issues you're happiest with; they define the
target voice.
